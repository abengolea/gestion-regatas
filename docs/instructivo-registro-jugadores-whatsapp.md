## Versión corta (para mensaje rápido)

```
⚽ *Escuelas River* — Registro de jugadores

1️⃣ Registrate: [TU_URL_APP]/auth/registro
2️⃣ Esperá la aprobación del entrenador/admin
3️⃣ Iniciá sesión: [TU_URL_APP]/auth/login
4️⃣ Completá tu perfil (nombre, apellido, fecha nac., tutor, email, foto)
5️⃣ Subí tu ficha médica (PDF) en Mi perfil → Ficha médica

Con el perfil completo tenés acceso a evaluaciones, videos, pagos y más.
```

---

## Cómo usar este instructivo

1. **Reemplazá `[TU_URL_APP]`** con la URL real de tu aplicación, por ejemplo:
   - Producción: `https://tu-dominio.com` o `https://escuelariver.com`
   - Staging: `https://staging.tu-dominio.com`

2. **Ejemplo de link final:** `https://tu-dominio.com/auth/registro`

3. **Enviá por WhatsApp** el bloque de texto completo o la versión corta según el espacio y el público.
